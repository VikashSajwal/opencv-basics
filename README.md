# OpenCV Basics – Image Reading

This project demonstrates basic image processing using OpenCV and Python.

## Features
- Read image from disk
- Display image using OpenCV
- Handle missing file errors
- Convert image to grayscale


## Files
- read_image.py – Reads and displays an image
- test.jpg – Sample image
- grayscale.py – Converts image to grayscale


## Technologies
- Python
- OpenCV
